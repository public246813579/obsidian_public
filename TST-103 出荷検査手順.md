# TST-103: 出荷検査手順

全台検査。出荷前のXY精度を [[PRT-109 出荷検査治具]] で検証。

## 手順

1. [[PRT-109 出荷検査治具]] をステージ上にセット
2. 固定ピンを [[PRT-101 リニアガイド]] 基準面に押し当て
3. ダイヤルゲージで9点測定
4. 判定基準: [[SPEC-101 ピック配置精度]] 準拠

## [[ECR-101 リニアガイド ベンダー変更]] 影響

B社ガイドは取付面高さが 0.3mm 高い → **[[PRT-109 出荷検査治具]] の固定ピンが接触不良**。
検査不能 → 出荷停止リスク。

## 発覚経緯

検査部が [[MTG-2026-04-06 週次定例]] で報告（ECR承認の3週間後、量産寸前）。
